# Keybindings

> [!NOTE]
> This list does NOT have to up to date and these are essentials only. If you want up to date experience, you should check [lua/keys.lua](../lua/keys.lua).

|Key|Description|
|-|-|
|`SPACE` `SPACE`|Telescope find file|
|`SPACE` lg|Telescope live grep|
|`SPACE`||
|`SPACE` e|Toggle file manager|
|`CTRL` `\`|Toggle terminal|
