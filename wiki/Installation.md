# Installation
Just clone this repo to your `~/.config/nvim` directory with this command;
```
git clone https://github.com/UmutKzl/kzlvim.git ~/.config/nvim
```

And launch Neovim with `nvim` command.

Don't forget to do `:checkhealth` command.

I recommend you to use a Nerd Font, spesifically JetBrainsMono Nerd Font.
